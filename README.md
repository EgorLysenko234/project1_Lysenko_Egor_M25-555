# Лабиринт сокровищ

Текстовая приключенческая игра, в которой игрок исследует лабиринт, решает загадки, собирает предметы и пытается найти выход.

## Установка

1. Убедитесь, что у вас установлены Python 3.12+ и Poetry
2. Клонируйте репозиторий
3. Установите зависимости:

bash
poetry install

4. Запустите игру командой:

bash
poetry run project
